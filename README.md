# QA Engineer Portfolio

A professional, responsive portfolio website for Yoga Wayan Pratama, Quality Assurance Engineer.

## Technologies Used
- React (Vite)
- Tailwind CSS

## Getting Started

To run the project locally:

1. Clone or download the repository.
2. Open your terminal and navigate to the project directory:
   \`\`\`bash
   cd qa-portfolio
   \`\`\`
3. Install the dependencies:
   \`\`\`bash
   npm install
   \`\`\`
4. Run the development server:
   \`\`\`bash
   npm run dev
   \`\`\`
5. Open your browser and navigate to the local server URL provided (usually `http://localhost:5173`).

## Project Structure
All the components (Navbar, Hero, About, Skills, Experience, Projects, QASamples, AutomationPortfolio, Contact, Footer) are located in `src/components/`.

## Customization
To update content, open the specific component file in `src/components/` and modify the text or properties. Code comments and structured data arrays make it easy to manage data without deep dive into the styling.
